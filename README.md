# Tock OS Website

[http://www.tockos.org/](http://www.tockos.org/)


## Building

Dependencies:

        sudo apt install ruby ruby-dev ruby-bundler
        sudo gem install jekyll jekyll-paginate jekyll-mentions


To build the website:

		jekyll serve


Now browse to http://localhost:4000 to view
