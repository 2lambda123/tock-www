Module kernel::hil::symmetric_encryption [−] [src]

Interfaces for accessing encryption and decryption of symmetric ciphers.

Only AES-128-ctr supported at the moment.

The interface is supposed to work for hardware supported crypto but should work for software implemented crypto as well.

State Machine:

  1. init()
  2. set_key()
  3. aes128_crypt_ctr(): can be used arbitrary number of times.
