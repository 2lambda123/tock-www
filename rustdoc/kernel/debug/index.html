Module kernel::debug [−] [src]

Provides a debug! macro for in-kernel debugging.

This module uses an internal buffer to write the strings into. If you are writing and the buffer fills up, you can make the size of output_buffer larger.

Example

debug!("Yes the code gets here with value {}", i);
